# Journal des sessions — Making Of

Ce dossier contient un fichier par session de travail, généré automatiquement par `/session-end`.

Chaque fichier inclut :
- Date et commits de la session
- Fichiers modifiés (stats)
- Résumé narratif de ce qui a été fait
- Moments forts / décisions notables
- Screenshot du jeu à l'état actuel

## Format des fichiers

`YYYY-MM-DD_session.md` — un fichier par jour de session.
Si plusieurs sessions le même jour, suffixe `_2`, `_3`, etc.

## Utilisation pour la vidéo

Ces journaux servent de matière première pour :
- Le script/narration de la vidéo making-of
- Identifier les moments visuellement intéressants
- Reconstruire la timeline du développement
- Les avant/après à chaque étape
