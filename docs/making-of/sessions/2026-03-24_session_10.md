## Ce qui a été fait

Session courte et chirurgicale : le jeu parle maintenant anglais.

L'objectif était simple mais structurant — **sortir toutes les chaînes
hardcodées** du code JS et les centraliser dans deux fichiers JSON, un
par langue. Le tout sans framework, sans TypeScript, sans rechargement
de page.

**E1 — Le module I18n.js** (68 lignes) : un singleton léger avec trois
méthodes essentielles. `t(key, params)` résout une clé dotted comme
`'tutorial.modes.plombee'` et remplace les variables `{n}`. `ta(key)`
retourne un tableau (pour les 16 tips de chargement). `detect()` lit
d'abord localStorage, puis `navigator.language`. Fallback FR automatique
si une clé est absente en EN — le jeu ne se casse jamais.

**E2 — fr.json / E3 — en.json** : 132 lignes chacun. Tous les menus,
les 16 tips de l'écran de chargement, les 4 phases du tutoriel, les
hints terrain, les labels in-game. Côté anglais, la terminologie FIPJP
officielle : Jack (cochonnet), End (mène), Iron Shot (tir au fer),
Rolling/Half-Lob/Lob (les modes de lancer), Composure (sang-froid).
**Galets et Carreau restent tels quels** — c'est de la couleur locale,
pas de la localization.

**E4 — Intégration dans 6 fichiers** : Le gros du travail. Chaque
`'APPUYEZ SUR ESPACE'`, `'Chargement...'`, `'Compris !'`, `'MENE 1'`
remplacé par son appel `I18n.t()`. Dans TitleScene, un petit bouton
`[FR|en]` / `[fr|EN]` apparaît en bas à droite — un clic, la scène
redémarre dans l'autre langue. Pas de rechargement de page.

## Moments forts

- **main.js passe de 3 à 5 lignes** — le `await I18n.load()` avant
  `new Phaser.Game(config)` est élégant. Phaser ne démarre que quand
  la langue est prête. Aucun "flash" de texte en français avant
  traduction.

- **Les terrain hints sont maintenant dynamiques** : au lieu d'un objet
  hardcodé `TERRAIN_HINTS = { colline: '...', docks: '...' }`, on fait
  `I18n.t('terrain_hints.colline')`. Plus de magic strings éparpillées.

- **La règle "Galets conservés"** : on aurait pu traduire en "Pebbles"
  ou "Coins". On a résisté. Galets c'est Galets — c'est l'âme du jeu.

## Décisions notables

- **top-level await dans main.js** : Vite/ESM le supporte nativement.
  Alternative rejetée : charger I18n dans BootScene.create() — mais
  alors les tips de l'écran de chargement auraient été en français pour
  les anglophones. On a préféré le chargement au plus tôt.

- **ta() pour les tableaux** : `t()` avec une clé qui pointe vers un
  array retournerait l'array brut, puis `.replace()` planterait. Ajout
  d'une méthode dédiée `ta(key)` — 4 lignes, propre, sans ambiguïté.

- **Sprint F prévu** : les fichiers JSON de données (characters.json,
  arcade.json, shop.json...) ne sont PAS touchés dans ce sprint. Les
  noms de personnages, barks, descriptions de boules — c'est le sprint
  suivant. Tout est déjà prévu dans le plan.
